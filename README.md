# Python Foundations: Daily Assignment

## Overview

This repository contains solutions to the Level 1, 2, and 3 tasks from the Python Foundations assignment.

### Tasks Solved

1. **Level 1**: **String Reversal** (`reverse_string.py`) - A script that takes a user's string input and outputs the reversed version.
2. **Level 2**: **Fibonacci Sequence Generator** (`fibonacci.py`) - A program that generates a Fibonacci sequence up to the nth term specified by the user.
3. **Level 3**: **CLI Tic-Tac-Toe Game** (`tic_tac_toe.py`) - A fully functional, two-player command-line based game with win and draw detection.

## How to Run

### Prerequisite



- Python 3.x installed on your machine.

### 1. String Reversal

Run the following command:

```bash
python reverse_string.py
```

**Follow the prompt** and enter any string to see its reversed counterpart.

### 2. Fibonacci Sequence Generator

Run the following command:

```bash
python fibonacci.py
```

**Follow the prompt** and enter the number of terms (e.g., 10) you want to generate.

### 3. Tic-Tac-Toe

Run the following command:

```bash
python tic_tac_toe.py
```

**Gameplay**:

- Two players take turns (Player X and Player O).
- Choose a position (1-9) to place your marker on the board.
- The game validates inputs and checks for a winner or a draw after each move.

## Project Structure

- `reverse_string.py`: Level 1 solution.
- `fibonacci.py`: Level 2 solution.
- `tic_tac_toe.py`: Level 3 solution.
- `README.md`: Project documentation and instructions.
